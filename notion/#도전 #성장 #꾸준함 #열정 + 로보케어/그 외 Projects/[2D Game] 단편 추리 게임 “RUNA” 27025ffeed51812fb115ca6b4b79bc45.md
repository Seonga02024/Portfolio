# [2D Game] 단편 추리 게임 “RUNA”

상태: 2D Game

![runaimg.png](%5B2D%20Game%5D%20%EB%8B%A8%ED%8E%B8%20%EC%B6%94%EB%A6%AC%20%EA%B2%8C%EC%9E%84%20%E2%80%9CRUNA%E2%80%9D/runaimg.png)

**소개**

- 한 소녀가 꿈에 갇히면서 꾸게 된 악몽을 탈출하기 위해 퍼즐을 풀면서 방을 이동하다가 탈출하는 2D 단편 추리 게임.

**기술 스택**

- Unity, C#

**업무 기간 및 팀 구성**

- 2023.2-3 (약 1개월)
- 디자이너 1명 , Unity 개발자 2명
- 클라이언트 개발 진행 (전체 기여도 50 %)

**역할 및 성과**

- 전반적인 게임 컨셉, 스토리, 퍼즐 공동 기획
- 2번째, 5번째 방 **퍼즐 구성 및 알고리즘 제작**
- **싱글 톤을 활용한 캐릭터 대사 창, 시스템 알림 창을 간단하게 사용할 수 있도록 제작**
- 사용자 소지품 저장 및 삭제 기능을 제작해 퍼즐에 활용할 수 있도록 구현 (Narration, Item 등)
- 최대한 코드 객체화
