- 라인 차트보다는 직관적이나 여전히 겹쳐서 안 보이는 영역과 시각적으로 파악하기 어려움

---

**3차 제작 :**

위 그래프들로는 3가지 정보(항목, 연도, 판매 수)를 동시에 보여주기 어려운 것 같아 → 히트맵(Heatmap)을 사용 변경 → 가로축은 '연도', 세로축은 '플랫폼' '장르'로 하고, 각 연도의 판매량을 색상의 농도로 표시

![genre_popularity_heatmap.png](2%20%EC%B0%A8%20%EB%8D%B0%EC%9D%B4%ED%84%B0%20%EC%8B%9C%EA%B0%81%20%ED%99%94%20%EA%B2%B0%EA%B3%BC%EB%AC%BC%20%EA%B3%B5%EC%9C%A0/genre_popularity_heatmap.png)

![platform_sales_heatmap.png](2%20%EC%B0%A8%20%EB%8D%B0%EC%9D%B4%ED%84%B0%20%EC%8B%9C%EA%B0%81%20%ED%99%94%20%EA%B2%B0%EA%B3%BC%EB%AC%BC%20%EA%B3%B5%EC%9C%A0/platform_sales_heatmap.png)

문제점 : 

- 어떤 연도에 어떤 플랫폼이나 장르가 특별히 '뜨거웠는지' 한눈에 파악 가능
- 전체적인 트렌드의 변화가 직관적임
- 라인 차트나 영역 차트와는 전혀 다른 관점에서 데이터를 탐색 가능
- 하지만, 인기가 많을수록 농도가 진하고 없을수록 농도가 연하면 좋겠음
- 색상 또한, 더 직관적으로 한 계열에서만 나타나면 좋겠음

---

**4차 제작 :**

낮은 값은 연한 색, 높은 값은 진한 색 으로 표현 수정 → 판매량이 거의 없는 곳은 흰색에 가깝게, 판매량이 폭발적인 곳은 아주 진한 파란/초록색으로 표시

![genre_popularity_heatmap_v2.png](2%20%EC%B0%A8%20%EB%8D%B0%EC%9D%B4%ED%84%B0%20%EC%8B%9C%EA%B0%81%20%ED%99%94%20%EA%B2%B0%EA%B3%BC%EB%AC%BC%20%EA%B3%B5%EC%9C%A0/genre_popularity_heatmap_v2.png)

![platform_sales_heatmap_v2.png](2%20%EC%B0%A8%20%EB%8D%B0%EC%9D%B4%ED%84%B0%20%EC%8B%9C%EA%B0%81%20%ED%99%94%20%EA%B2%B0%EA%B3%BC%EB%AC%BC%20%EA%B3%B5%EC%9C%A0/platform_sales_heatmap_v2.png)

문제점 : 

- 인기 정도 / 전반적인 흐름 파악이 매우 잘됨
- 마지막으로, 가장 높았던 인기 도에 표시와 제목을 한국어로 바꾸고 표와 띄우기 간격을 주고 싶음
