# [Study/Game] Python Study - 데이터 시각화 및 AI 게임 제작 프로젝트

![image.png](%5BStudy%20Game%5D%20Python%20Study%20-%20%EB%8D%B0%EC%9D%B4%ED%84%B0%20%EC%8B%9C%EA%B0%81%ED%99%94%20%EB%B0%8F%20AI%20%EA%B2%8C%EC%9E%84%20%EC%A0%9C%EC%9E%91%20%ED%94%84%EB%A1%9C%EC%A0%9D/image.png)

- **스터디 주제:**

       AI를 베이스로 파이썬을 활용한 데이터 시각화 및 간단한 게임만들기

- **목표:**

       실무에 AI를 똑똑하고 적극적으로 잘 활용하기 위해, 엑셀, 재무 데이터 등을 시각화 및 분석화 하는 방법을 스터디하고 python 및 코드에 대해 친숙하게 느껴질 수 있도록 간단한 게임 만들기를 같이 진행하여 최종, 각자만의 결과물을 갖고 공유 할 수 있는 스터디 참여

- **참여 기간:** 2025 년 8월 1일 ~ 10월 17일 (약 3달)
- **참여 역할 :** 팀원(서포트)

**1 ~ 3 주차 : python 언어 스터디 및 기초 데이터 시각 화 공부 진행**

**4 주차 : 1차 각자 데이터 시각 화 결과물 공유**

[google_study_4.pptx](%5BStudy%20Game%5D%20Python%20Study%20-%20%EB%8D%B0%EC%9D%B4%ED%84%B0%20%EC%8B%9C%EA%B0%81%ED%99%94%20%EB%B0%8F%20AI%20%EA%B2%8C%EC%9E%84%20%EC%A0%9C%EC%9E%91%20%ED%94%84%EB%A1%9C%EC%A0%9D/google_study_4.pptx)

결과물 : 용의 둥지라는 맵(임의의)에서 발생한 유저들의 사망 위치 좌표 데이터로 특정 지역에서 사망이 집중되는지 분석하여 맵 밸런스를 조정하는 데 사용하기 위한 히트맵 제작 → 히트맵으로 시각화하면 사망이 잦은 위험 지역을 직관적으로 파악 → 현재 사용 중인 사각형 격자 형태의 히트맵 대신, sns.kdeplot을 사용하면 데이터의 밀도를 부드러운 등고선형태로 표현 가능 : 더 세련되고 분포를 직관적으로 파악하는 데 도움 → 게임 맵(AI)을 오버레이 : 위험 지형 파악을 더 원활히 가능
