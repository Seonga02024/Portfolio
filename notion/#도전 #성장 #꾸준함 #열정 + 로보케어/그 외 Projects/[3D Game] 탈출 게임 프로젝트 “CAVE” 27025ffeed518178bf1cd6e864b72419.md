# [3D Game] 탈출 게임 프로젝트 “CAVE”

상태: 3D Game

![1.PNG](%5B3D%20Game%5D%20%ED%83%88%EC%B6%9C%20%EA%B2%8C%EC%9E%84%20%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8%20%E2%80%9CCAVE%E2%80%9D/1.png)

**소개**

- 대학생 1학년 때, 게임 프로그래밍 수업 프로젝트로 개발한 간단한 게임
- CAVE에서 탈출하기 (1인칭 시점 / 3D 게임)
- 조난당한 플레이어가 동굴에서 금괴, 열쇠, 도망치는 몬스터를 잡아서 탈출하는 단편 게임
- 쉽게 탈출하지 못하도록, 돌아다니는 몬스터가 존재하며 플레이어가 발각 당할 시 쫓아옴

**기술 스택**

- Unity, C#

**업무 기간**

- 2019.05 ~ 2019.06 (약 2개월)

**역할**

- 기획 및 개발 (전체 기여도 100 %)

**배운 점**

- **사용자 HP 시스템** 및 아이템 소지 UI 구현
- 몬스터 **공격 방법**에 대해 기획하고 **수학 식을 활용하여 구현**
- 다른 몬스터의 **도망 방법**에 대해 기획하고 **수학 식을 활용하여 구현**
