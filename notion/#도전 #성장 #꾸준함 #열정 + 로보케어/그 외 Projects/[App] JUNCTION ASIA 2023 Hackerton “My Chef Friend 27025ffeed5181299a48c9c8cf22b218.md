# [App] JUNCTION ASIA 2023 Hackerton “My Chef Friend” App

상태: APP

![Untitled](%5BApp%5D%20JUNCTION%20ASIA%202023%20Hackerton%20%E2%80%9CMy%20Chef%20Friend/Untitled.png)

**소개**

- **사용자 맞춤형 AI 친구와 함께하는 JAKA의 로봇 요리 배달/주문 서비스 앱**으로 사용자는 앱을 사용하면서 가상의 AI 친구를 만나게 되고 이 친구에게 밥을 주고 대화를 하는 등의 데일리 활동과 음식을 주문하는 과정, 다양한 미션등에서 친밀도를 쌓을 수 있습니다. 이 친밀도에 따라서 주문을 하는 과정에서 추가적인 옵션이 해금되거나 각 지점별로 호감도 랭킹을 보여주거나, 포인트로 결제를 하거나 악세서리를 사는 등 사용자의 취향에 따라 AI 친구를 꾸며 친밀도를 증가시켜 앱과 해당 앱을 이용한 지점/브랜드의 인지도를 상승 시키고자 합니다.

자세한 기획서 : [My Chef Firend App 기획서](https://www.notion.so/My-Chef-Firend-App-be7608792d0b46e9a3d3bd9f7426c00d?pvs=21) 

**기술 스택**

- Unity, C#, redis, mysql, rest api

**업무 기간**

- 2028.3.18 ~ 2023.8.20 (약 2일)

**역할**

- 총 5명이 함께 진행 (개발 3명(프론트 2 / 백엔드 1), 디자이너 1명, 기획 1명)
- 클라이언트 개발 진행 (전체 기여도 20 %)
- 전체 front App 개발 + sever 연결

**성과**

- **Unity WebGL** 구현
- **Rest API** 공부 및 구현 (**http 통신 및 서버 연결**)
- 서버와 **REST API 통신하여 json 데이터 받아서 활용**하기
- **access token을 이용하여 서버와 인증인가 방식 구현**
- **ChatGPT를 이용한 인공지능 대화 기능 구현**
- 기획자, 백엔드 개발자, 디자이너와의 협업 경험
