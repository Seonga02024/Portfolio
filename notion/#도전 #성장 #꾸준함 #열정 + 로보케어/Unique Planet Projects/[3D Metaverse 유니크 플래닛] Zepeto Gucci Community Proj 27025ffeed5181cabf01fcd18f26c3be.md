# [3D Metaverse/유니크 플래닛] Zepeto Gucci Community Project

상태: 3D Game

![Untitled](%5B3D%20Metaverse%20%EC%9C%A0%EB%8B%88%ED%81%AC%20%ED%94%8C%EB%9E%98%EB%8B%9B%5D%20Zepeto%20Gucci%20Community%20Proj/Untitled%201.png)

**소개**

- 메타버스 플랫폼인 Naver Zepeto 내에 Gucci와 콜라보해서 출시한 월드입니다. 해당 콘텐츠 내에서 다수의 사람들이 모여서 소통하고 Gucci 관련된 옷과 가방을 직접 자신의 캐릭터에 착용하고 구매할 수 있습니다. 또한, Gucci 패션쇼를 실시간 스트리밍을 진행하기도 하였으며 관련 홍보 영상을 시청할 수 있습니다.

**기술 스택**

- Unity, TypeScript

**업무 기간**

- 2023년 8월 28일 - 9월 29일 : 월드 개발
- 2023년 9월 29일 - 10월 6일 : 출시 및 유지 보수 기간
- 2023년 12월 24일 : 크리스마스 업데이트
- 이후 패션쇼 업데이트, 상품 업데이트 진행

**역할**

- 클라이언트 개발 (전체 기여도 100 %)
- 업데이트 관련 개발 (전체 기여도 50%)

**성과**

- Unity와 TypeScript 을 이용해 개발하였으며 **제페토 API 기술 문서를 읽고 활용**해 제페토 내 콘텐츠를 제작할 수 있었습니다. 이를 통해 Gucci 월드를 성공적으로 출시했으며 **디자이너와 소통**을 통해 **사용자 경험을 향상시키기 위한 기능**들을 정확하게 파악하고 구현했습니다.
- 멀티플레이 환경을 위한 **서버와 클라이언트 간의 통신 방식을 이해**하고 구현하여, 많은 유저들이 원활하게 상호작용하고 소통할 수 있는 환경을 제공했습니다. 또한, 실시간 라이브와 출시 이후 이루어지는 **유지 보수와 보완 작업을 경험하며 실무에 대한 역량**을 키울 수 있었습니다.

**프로젝트 성과**

- 2024년 5월 기준 누적 방문 수 882K 이상
- 실제 gucci 패션쇼 라이브 방송 2회 진행
- 현재까지 시즌 별 이벤트, 콘텐츠 업데이트 진행
- 콘텐츠 내 전시 아이템 수익 창출
- 추후에도 콘텐츠 업데이트 예정
