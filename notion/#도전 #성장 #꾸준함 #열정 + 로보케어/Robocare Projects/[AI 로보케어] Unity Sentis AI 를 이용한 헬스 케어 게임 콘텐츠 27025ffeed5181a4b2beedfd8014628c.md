# [AI/로보케어] Unity Sentis AI 를 이용한 헬스 케어 게임 콘텐츠

**현 상황 : 2026 구글 플레이 스토어 상용화 계획**

## 제작 배경 및 환경

- 사용 기술 및 언어 : Unity + C#
- 업무 기간 : 2025년 7월 ~ 10월
- 역할 : 3인 개발
- 프로젝트 소개 :

## 게임 개요

- **장르** : 체조, 게임 복합 콘텐츠
- **플랫폼** : 별도 게임 앱
- **플레이어 수**  : 1인 플레이
- **목표** :  체조를 통해 자세 개선 및 운동을 하며 게임을 통해 재미와 동시에 운동을 돕는

## 게임 메커니즘

1. 플레이 시간 :  약 5분
2. 시점 : 1 인칭

## 개발하면서 신경 쓴 부분

실시간 카메라 영상 입력을 이용하여 사용자의 상체 움직임을 분석

모션 정확도보다는 동기 부여와 재미에 초점을 둔 고령자 친화형 UI

리듬 타이밍에 맞추어 운동 동작을 수행하는 콘텐츠 개발

## **관련 성과 :**

체조 콘텐츠의 몰입도를 크게 향상

고령자 테스트 환경에서 사용자 피드백 긍정적
