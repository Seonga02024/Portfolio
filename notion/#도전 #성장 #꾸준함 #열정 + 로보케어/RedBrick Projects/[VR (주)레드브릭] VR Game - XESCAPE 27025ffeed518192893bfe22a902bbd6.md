## 제작 환경

- 사용 기술 및 언어 : Redbrick + JavaScript
- 업무 기간 : 2024년 11월 14일 출시 (약 2주 동안 개발)
- 역할 : 디자이너, 3D 모델러와 협업 **1인 개발**

## 개요

- **장르** : VR 퍼즐 게임
- **플랫폼** : 레드브릭
- **플레이어 수**  : 1인 (싱글 플레이)
- **목표** :  모든 퍼즐을 풀고 탈출해라 (엔딩을 보아라)
- **VR 기기 :** Meta Quest 3

## 게임 배경

- **장소**  : 아포칼립스 배경
- **특징** : 어둡고 기계적임.

## 게임 메커니즘

1. 플레이 시간 :  약 10분
2. 시점 : 1인칭

## 개발하면서 신경 쓴 부분

- 전반적인 퍼즐 기획에 기여 - 만들 수 있는 VR 퍼즐 예제 scene 만들어서 기획, 디자이너에게 제공함 (관련 링크 : https://redbrick.land/detail-play?pid=37f0ae87-4c49-46fb-8972-f5243ab3f733)
- 언제든 수정하고 추가하게 쉬운 구조로 만들기 / 각 게임 타이밍 구분하기 (스토리 시작 전, 스토리 시작 후, 스토리 끝, 게임 1 시작, 리프트 상승 시작, 끝, 게임 2시작 등)
- 각 퍼즐에 대한 상호작용이 잘 이루어지게 만들기 (터치 시, 소리, 물체 움직임 구상 등)
